app.use(express.static("./public")) : is middleware ke vajah se humara backend , frontend bhi dikhana chalu kar diya kyuki browser jab request kar raha tha backend ke index.html ke api pe toh , woh api humne create hi nhi ki thi , toh humne server ko program kiya tha ki agar aisi koi bhi request ki hai toh wild card ke through index.html send kar dena
